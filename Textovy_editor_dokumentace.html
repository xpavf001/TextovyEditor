Úvod

Tato semestrální práce z se zaobírá tvorbou XML dokumentu, který popisuje soubor kam ukládá fiktivní textový editor své nastavení. Součástí práce je vytvoření XSD schéma, XSL, XSL transformace do HTML a dokumentace. Vše je tvořeno pro fiktivní textový editor.

Metodika

Všechny dokumenty byly vytvořeny v programu Sublime Text.

Nejprve bylo vytvořeno schéma XSD se základnímy elementy, podelementy a atributy. Dále bylo třeba vytvořit restrikce na datové typy a provést celkovou korekci a úpravu struktury. Poté došlo k vytváření samotného XML dokumentu, který korespondoval s vytvořeným XSD. Tímto způsobem se lépe vytvářelo XML a dokument se lépe plnil smyšlenými daty.

Po vytvoření XML a XSL bylo třeba zajistit jejich validitu. Validita těchto dvou dokumentů byla zkontrolována pomocí webového validátoru . Zároveň bylo zkontrolováno, že XML dokument odpovídá XSD schématu.

XSL transformace do HTML byla psána ručně. Data jsou strukturována do HTML tabulek podle kategorií vyplývajících z designu XSD dokumentu. Obsahují stručně a přehledně zapsaná data obsažená v XML dokumentu. Při testování jsem zjistil, že prohlížeč Opera neumí s transformacemi pracovat, zobrazí pouze plain text. Pouze prohlížeč Internet Explorer zobrazil HTML transformaci správně. Transformace do HTML obsahuje všechna data z XML souboru.

Celý projekt byl v průběhu jeho tvorby postupně nahráván (a verzován) na platformě GitHub.

Popis transformace

HTML

Data jsou transformována do HTML dokumentu. Jsou strukturována tak, jak by typicky byly strukturovány možnosti nastavení textového editoru - na Základní nastavení, Formát dokumentu, Práci s písmem, Stylizaci, Ohraničení, a Číslování stránek. Jednotlivé elementy a atributy jsou zapsány do tabulek. Všechny elementy datového typu "boolean" jsou zabaleny v podmínkách, přes které se přepisují z 1/0 na Zapnuto/Vypnuto. Transformace od HTML obsahuje všechna data z XML souboru.

Závěr

Jazyk XML je používán pro strukturované ukládání dat nebo pro komunikaci mezi programy. Jeho výhodou je dobrá čitelnost, široké možnosti filtrování a omezování dat a datových typů. Práce s ním je také snadná, protože k jeho editaci stačí pouze obyčejný textový editor.
